# Synchrolog

## Installation

```bash
$ npm install git://github.com/synchrolog/synchrolog-node.git
```

## Usage

```js
const synchrolog = require('synchrolog-node');

// Initialize your express app, and pass the app to the synchrolog module
var app = express();
synchrolog(app, 'YOUR_API_KEY');
```
